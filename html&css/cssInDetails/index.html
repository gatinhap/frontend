Using CSS

  • Inline - by using the style attribute inside HTML elements:
    • <p style="color:red;">A red paragraph.>
  • Internal - by using a style element in the head section:
    • <head>
    <style>
    body {background-color: powderblue;}
    h1 {color: blue;}
    p {color: red;}
    <style>
    <head>
  • External - by using a link element to link to an external CSS file:
    • <head>
    <link rel="stylesheet" href="styles.css">
    <head>

Basic CSS properties

  • Use the CSS border property for borders. They can be set with a specific width, style, and color, e.g.
    • p {
    border: 3px solid coral;
    }

    The default border is medium none color, where color is the current color of the element

    You can modify the corners of an element’s border box with the border-radius property and set the same curvature for each of them

  • Use the CSS padding property for space inside the border:
    • padding-top
    • padding-right
    • padding-bottom
    • padding-left
  • Shorthand property:
    padding: 25px 50px 75px 100px
    padding: 20px 10px
    20px: top and bottom; 10px: right and left
  • Use the CSS margin property for space outside the border
    Same logic as in case of padding apart from the fact that vertical margins collapse. The larger of the two vertical margins sets the distance between adjacent elements

The Box Model

The box model properties:

  • Width and height — specifies the width and height of the content area.
  • Padding — specifies the amount of space between the content area and the border.
  • Border — specifies the thickness and style of the border surrounding the content area and padding.
  • Margin — specifies the amount of space between the border and the outside edge of the element.
